About

What is it?

MyTopSongs is a tool that uses spotfy's api to display a users' top 10 songs from the last month, 6 months and all-time. It was inspired by 'receiptify' a popular tool to display a user's spotify statistics in a reciept format. It can be found here.

How does it work?

This project's code uses the open source spotify api to grab listening data from users and report their top songs. Unfortunately, with the data available, I'm limited to only the last month, half year and all time. I don't know how spotify stores any of this data.

How was it made?

This project's code uses python for the backend, mainly using the tool 'spotipy' to simplify the data retrieving and authorization process. HTML templates to display the necessary data, CSS for styling and Javascript for interactivity. The code can be found here.
